# SurfDesk release rule

This app is the independent GitHub repository `angusi01/surfdesk` and Render service `srv-d90h050k1i2s73fm0j5g`.

Push its complete intended source state to the repository's `main` branch, deploy only to that Render service, and verify `surfdesk.com.au` plus `www.surfdesk.com.au`. A `passive-portfolio` monorepo push is not a substitute for this app's repository push and deployment.

Treat SurfDesk as a simple one-shot booking/transaction app. It is not a vertical SaaS product, not a multi-tenant platform, and not a dashboard-first workflow. Prefer direct booking, payment, and confirmation flows consistent with the lighter-weight app model used elsewhere in this workspace.

The user grants standing authorization to create, use, and delegate to subagents or parallel agents for any task in this repository.
